Import from health apps

Apple Health (iPhone): Health app → your picture (top right) → Export All Health Data → you get export.zip. Extract the zip (tap it in the Files app, or copy it to this computer and unzip), then choose the export.xml inside it here.
Samsung Health: Samsung Health → ⋮ Settings → Download personal data, then choose the step_daily_trend….csv (steps) or weight….csv (body weight) file here.

Use on your phone

On this PC, double-click “Start OptimalFit (Phone).bat” in the optimal-fit folder. It starts the server in phone mode and prints the address and a 6-digit pairing code (needed once for the AI coach). Your phone must be on the same WiFi as this PC. If Windows asks about the firewall, allow Python on private networks.

Then on the phone, open the address shown and add it to your home screen:
iPhone: Safari → Share button → Add to Home Screen.
Android / Samsung: Chrome → ⋮ menu → Add to Home screen (or Install app).

Honest notes: your data lives in each device’s own browser storage — the phone starts empty; use Export on the PC and Import on the phone to move it. The AI coach only works while the PC server is running.
